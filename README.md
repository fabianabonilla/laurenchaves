# Lauren Chaves Portfolio

A polished GitHub Pages portfolio for an interior design student, built as a static site with editorial typography, large-format project imagery, process notes, and contact links.

## Customize

- Replace placeholder project names and descriptions in `index.html`.
- Swap the image URLs with Lauren's own renderings, drawings, models, and finished work.
- Update contact links in the contact section.

## Publish on GitHub Pages

In the repository settings, enable Pages from the `main` branch and root folder. The site will serve as a static portfolio.
