Studio notes

A portfolio built like a room: entry, rhythm, reveal.

Every project begins with a feeling, then becomes a plan: site constraints, user rituals, circulation, finishes, and light. The result is work that feels polished but still shows the hand behind it.

Process

From first impression to final presentation.

01

Read the client

Interviews, rituals, reference imagery, and use patterns become the brief.

02

Shape the plan

Adjacencies, circulation, scale, and furniture layouts define the room logic.

03

Layer the feeling

Materials, lighting, color, and texture build the emotional temperature.

04

Tell the story

Renderings, boards, drawings, and details make the work easy to understand.
